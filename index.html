Introduction

Welcome to my portfolio!
I’m a former kindergarten teacher turned software engineer with a passion for building thoughtful, user-centered web applications. After transitioning into tech, I earned my Bachelor of Science in Software Engineering and built a strong foundation in full-stack development using tools like Python, JavaScript, HTML/CSS, and SQL.
My background in education has equipped me with strengths in adaptability, clear communication, and creative problem-solving — all of which I bring to the world of software development. I recently completed a wellness journal web application as part of my capstone, demonstrating my skills in frontend design, backend logic, and cloud deployment.
I’m especially drawn to opportunities that involve user experience, accessibility, and purposeful design — areas where my ability to think empathetically and iteratively truly shine. I’m currently seeking an entry-level software engineering role where I can grow, collaborate, and contribute to impactful digital experiences.
Let’s connect — I’m excited to continue building and learning alongside a great team.

Projects

Random Quote Machine Website

This website was created for the freeCodeCamp.com "Frontend Development Libraries Certification. It allows the user to "Generate a Random Quote" by pressing the button and was my first time using React to complete a project!

Wellness Journal Application

This full-stack web app was developed for my Software Engineering Capstone project. Built with Flask, SQLite, HTML, CSS, and JavaScript, it allows users to log and track daily fitness, hydration, mood, goals, sleep, and reflections. Designed for health-conscious users, the app supports CRUD operations, data validation, and secure login functionality. The project was developed using Agile practices and deployed using Render.
